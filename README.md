# 📊 E-Commerce Sales Performance Dashboard

## 🔹 Overview
This project is a Tableau dashboard designed to analyze sales performance using interactive visualizations.

## 🔹 Features
- Monthly sales trend analysis
- Category-wise and region-wise performance
- Top 10 customers identification
- KPI cards (Total Sales, Orders, Customers)

## 🔹 Tools Used
- Tableau
